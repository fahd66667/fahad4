Software

Dev Desk // Berlin

Software is no longer written; it is grown. Using the latest Large Action Models, developers can now describe a problem, and the OS builds a custom, single-use application to solve it in real-time. This is the end of the "App Store" as we know it, replaced by a fluid, generative digital environment.

Fluid Interfaces

Your phone is no longer a collection of icons. It is a single, unified agent that manifests whatever tool you need in the moment. If you are traveling, the interface becomes a navigator; if you are cooking, it becomes a sous-chef. The app as a product is dead; the app as a service is the new reality.
